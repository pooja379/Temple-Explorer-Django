🛕 Temple Explorer India

Discover the rich spiritual heritage of India through an interactive digital journey.

Our Mission

Temple Explorer India is a modern Django web application developed to help devotees, pilgrims, and travelers discover India's rich spiritual heritage. The platform allows users to explore temples using an interactive map, search by temple name, city, or deity, view detailed temple information, read history, timings, festivals, and architecture, submit reviews, check average ratings, save favorite temples, securely register and log in, and navigate directly using Google Maps. The project also includes REST API endpoints built with Django REST Framework, making it ready for future integration with mobile applications and modern frontend frameworks. Our mission is to preserve India's spiritual heritage through technology while making temple information easily accessible to everyone.
